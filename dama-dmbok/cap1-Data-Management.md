
1. INTRODUÇÃO:

- Tratar dados como um ATIVO da empresa

- Dentro de uma empresa, Data Management deve incluir:
    - Todo o suporte aos stakeholders
    - Capturar, armazenar, proteger, garantir a integridade
    - Garantir a qualidade
    - Garantir que a privacidade e confidencialidade
    - Previnir o acesso e manipulação inapropriados dos dados


2. Essential Concepts

- Data:
    - Oq são dados mudou com o tempo
    - Mesmo dentro de uma unica empresa existem varias maneiras de representar uma mesma ideia, um mesmo dado
    - O problema da multiplicidade se multiplica, por isso "industrializar" o dado ajuda trazer consistencia aos dados
